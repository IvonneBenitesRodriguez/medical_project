### Medical Project Frontend <a name="about-project"></a>

medical_project repository is built using programming language named JavaScript but specifically in this case We used a framework named **React**.

This website is **an ecommerce** for offering chocolates and cakes as gifts to users in any special occassion.

We used also **for the payment the stripe platform**.
